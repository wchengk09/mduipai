# 新一代对拍器(多线程)

## 用法

### 编译

此对拍器废除了`./comp`和`./compall`脚本，而是使用`Makefile`自动编译。

你只需要运行`make`命令，就可以自动编译`a`,`b`,`rand`。

### 对拍

运行命令：`./main (线程数) (时间限制/ms) (内存限制/MB)`。

这 $2$ 个参数（线程数、时间限制、内存限制）都是可选的。

默认情况下，线程数为 `4` ，时间限制为`1000ms`，内存限制为`256MB`。

## 新特性

+ 可以限制程序的运行时间和内存。
+ 可以实时显示对拍信息。
+ 支持多线程。
